# Python
Практическая работа анализ данных резюме hh.ru | Data Science проект

Описание проекта

Цель: Построить модель предсказания желаемой заработной платы соискателей HeadHunter на основе их характеристик.

Исходные данные: 44 744 резюме с hh.ru (файл [dst-3.0_16_1_hh_database.csv](https://drive.google.com/file/d/1UuzlajwpheBPP1LrpD909CKwqS6JEkk1/view?usp=sharing))

Проблема: большинство соискателей не указывают конкретную ЗП, следовательно рекомендательная система HH работает неэффективно.

Этапы работы
1. Базовый анализ структуры данных
2. Преобразование данных
3. Разведывательный анализ (EDA)
4. Очистка данных

Выводы и рекомендации:
Данные представлены высокого качества — 99.9% заполнены, имеют реалистичные значения
Мощными предикторами являются: опыт работы, город проживания, мобильность (готовность к переезду и командировкаам), график работы

Следующие шаги: Обучение модели, валидация, feature importance.
